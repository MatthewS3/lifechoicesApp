Express Example

General Routing:
Path HTTP Methods Description
/ GET Display the Home Page
You must log in first.

User Routing:
Path HTTP Methods Description
/Users/ GET Get all the users
/User/:id GET Fetch a specific User
/User/register POST Register a new User
You must log in first.

Product Routing:
Path HTTP Methods Description
/Product/ GET To display all products
/Products/:id GET Fetch a specific product
/Products/addProducts GET Add a new Product
You must log in first.
